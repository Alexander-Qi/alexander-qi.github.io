一些好用的在线数据库及在线工具

发表于 2020-03-18

本文介绍计算化学相关的一些实用的数据库和在线工具。本文仅挑选一些内容较佳，访问体验较好的在线数据库和工具；若需要了解更多，可以参考《计算化学相关的免费的在线数据库及工具》。本文会不断更新。

Basis Set Exchange 包含：基组、赝势 简介：可以说是目前最重要、最全面的基组+赝势数据库，包含绝大多数（特别是常用的）目前已经发表的基组和赝势。

TURBOMOLE basis set library II 包含：基组、赝势 简介：包括def、def2系列基组。除了普通基组，还可以获得/J、/JK、/C密度拟合基组。

Energy-consistent Pseudopotentials of the Stuttgart/Cologne Group 包含：基组、赝势 简介：Stuttgart赝势+赝势基组数据库，虽然BSE上也有Stuttgart赝势和配套基组的定义，Gaussian等程序也自带了，但都比较老，只有这里的才是最新、最全的。

ChemSpider 包含：分子平面结构、实验测定的和由ACD/Labs、EPISuite、ChemAxon软件预测的理化性质（LogP、LogD(PH=5.5和7.4时)、水溶性、分子体积、密度、沸点、闪点、蒸汽压、气化焓、折光率、可极化率、表面张力、SASA等），以及毒性、分子简介、Smile/InChI/InChIKey字符串、在其它分子数据库中的编号和链接、相关文章及专利、同义词、相关蛋白质、NMR/IR光谱图 简介：众多的在线分子数据库的信息整合，便于用户搜索，数据来自200种数据库。

Spectral Database for Organic Compounds, SDBS 包含：有机化合物光谱（EI-MS、FT-IR、H-NMR、C13-NMR、ESR、Raman） 简介：含3万余个化合物，其中以商业化学试剂为主，约2/3的数据是6碳至16碳的化合物。数据大部分是其自行测定的，并不断添加。

NIST Chemistry WebBook 包含：分子量、CAS登记号、热力学数据、红外、紫外、气象色谱、质谱 简介：美国国家标准与技术研究院NIST的基于Web的物性数据库。

NIST Chemical Kinetics Database 包含：化学动力学数据、指前因子、活化能等 简介：包含38000个基元反应的的化学动力学数据。

Computational Chemistry Comparison and Benchmark DataBase 包含：方法、泛函、基组、比较、测评 简介：包括各种量子化学方法、各种基组下对不同分子的各种属性的计算结果，也包含实验数据。可用来对比不同方法计算结果优劣。其中的频率计算校正因子尤其有用。

Crystallography Open Database 包含：无机、有机晶体结构 简介：免费的晶体学数据库，可以直接得到cif文件。

American Mineralogist Crystal Structure Database 包含：无机晶体结构 简介：可以获得大量无机晶体的晶体学信息和结构文件。

Virtual Computational Chemistry Laboratory 简介：在线版的Babel，可以支持几十种结构文件格式的转换。

WebMO Demo Server 简介：提供了友好的GUI界面，在线绘制或导入结构并调用服务器上的Gaussian、Gamess、Mopac、Molpro、NWChem、QChem、Tinker程序进行量化/分子力学计算。对于免费用户WebMO提供了guest帐户登入，但运算时间限制在60秒以内，可以用来学习使用这些量化程序。

Xemistry Sketcher & Internet Structure Data Retrieval Tool Demonstration 简介：非常好的在线绘制平面分子结构的工具，它的一大好处是不需要安装java运行环境或其它插件，也不必等待缓慢的载入过程，打开网页直接就可以用。绘制结构方便，界面简洁，还可以进入模版页面直接调用基团。在绘制过程中可以实时生成SMILE、InChI等字符串，可以自动将当前绘制的结构在化学结构数据库中检索，立刻得到它的比如CAS编号、PubChem CID编号、名称等信息。绘制的结构可以导出为mol、ChemDraw的CDX、Sybyl的sln等格式，还可以导出png/gif等图形、pdf文件、swf文件等。也可以直接输入Smile字符串，或者上传分子结构文件来导入此编辑器。
