# Jian-Qing Qi's Personal Homepage

Welcome to my personal homepage repository! This site is built using Jekyll and the [al-folio](https://github.com/alshedivat/al-folio) theme.

## About Me

I am a PhD candidate in physical organic chemistry at Tsinghua University under the supervision of Prof. Lei Jiao, expecting to graduate in June 2027. I have been conducting research within the [Jiao Research Group](https://www.jiaolei.group/) since 2018.

My research integrates advanced instrumental analysis with computational methods to investigate chemical reaction mechanisms in the fields of photochemistry and radical chemistry. My current focus is on radical-mediated processes, utilizing U-PSD TREPR to monitor transient open-shell intermediates.

### Technical Expertise
- **Chemical synthesis and mechanistic analysis**
- **Advanced spectroscopic characterization** (transient absorption, steady-state/transient EPR, U-PSD TREPR)
- **Quantum chemical calculations** (Gaussian, ORCA, xtb)
- **Programming and AI-assisted coding** (Python, MATLAB, Claude Code, Gemini CLI)

## Local Development

To run this website locally, ensure you have Docker installed and run the included script:

```bash
./run_local.sh
```

The site will be available at `http://localhost:8080`.
