免费SSH+SFTP软件：Bitvise SSH Client

发表于 2019-09-09 更新于 2019-09-08 分类于 resources

SSH (Secure Shell) 是目前较为可靠，专为远程登录会话和其他网络服务提供安全性的协议。

SSH协议使用加密来保护客户端和服务器之间的连接。所有用户身份验证，命令，输出和文件传输都经过加密，以防止网络中的攻击。有关SSH协议如何工作的详细信息，请参阅协议页面。要了解SSH文件传输协议，请参阅SFTP页面。

SSH.comSSH (SECURE SHELL)

SFTP技术能让我们更安全、更快捷地传输文件。

SFTP（SSH文件传输协议）是一种安全的文件传输协议。它运行在SSH协议上。它支持SSH的完整安全性和身份验证功能。

SFTP几乎取代了传统FTP作为文件传输协议，并且正在迅速取代FTP/S。它提供了这些协议提供的所有功能，但更安全，更可靠，配置更简单。基本没有理由再使用传统协议。

SFTP还可以防止密码嗅探和中间人攻击。它使用加密和加密哈希函数保护数据的完整性，并对服务器和用户进行验证。

SSH.comSFTP – SSH SECURE FILE TRANSFER PROTOCOL

安装完成后，Bitvise SSH Client共需要占据约133 MB的磁盘空间。

其主界面非常简洁大气。Bitvise SSH Client可以自动在profile中保存密码和凭证文件，设定好后直接点击“Log In”即可登录。

登录完后会自动打开两个窗口。

一个是基于xterm的终端界面，可以看到终端字体是Consolas，大小约为11~12。相较于Tectia，非常美观。

一个是SFTP界面，这个界面的操作和大多数其他的SFTP软件相似，可以调整窗口大小，对于Windows 10和高分屏都适配完整。

不像Tectia，你可以在Bitvise SSH Client中直接关闭终端和SFTP窗口，登出时只需要单击“Log Out”即可。
